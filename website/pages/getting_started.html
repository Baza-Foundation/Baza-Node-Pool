Baza Dedicated Miner

bazadedicatedminer is a high performance, open source, cross platform RandomX, KawPow, CryptoNight and AstroBWT unified CPU/GPU miner forked from xmrig. Official binaries are available for Windows, Ubuntu Linux and Debian.

Mining Backends

  • CPU (x64/ARMv8)
  • OpenCL for AMD GPUs.
  • CUDA for NVIDIA GPUs via external CUDA plugin.

Download

  • Binary releases

Build

Ubuntu and Debian

Basic Build

Basic build is good for local machine, because it is easy, but if you need to run the miner on other machines please take a look at advanced build. 

1. sudo apt-get install git build-essential cmake libuv1-dev libssl-dev libhwloc-dev
2. git clone https://gitlab.ekata.io/baza-foundation/baza-dedicated-miner.git
3. mkdir baza-dedicated-miner/build && cd baza-dedicated-miner/build
4. cmake ..
5. make -j$(nproc)
Advanced Build
1. sudo apt-get install git build-essential cmake automake libtool autoconf
2. git clone https://gitlab.ekata.io/baza-foundation/baza-dedicated-miner.git
3. mkdir baza-dedicated-miner/build && cd baza-dedicated-miner/scripts
4. ./build_deps.sh && cd ../build
5. cmake .. -DXMRIG_DEPS=scripts/deps
6. make -j$(nproc)

Running

  1. Download the binary from the release page
  2. Extract the zip and go to the directory
  3. Run with this command ./bazadedicatedminer -u <your_baza_wallet_address>

CUDA

Build

Ubuntu and Debian

Follow instructions on https://developer.nvidia.com/cuda-downloads to install CUDA. 

1. git clone https://github.com/xmrig/xmrig-cuda.git
2. mkdir xmrig-cuda/build && cd xmrig-cuda/build
3. cmake .. -DCUDA_LIB=/usr/local/cuda/lib64/stubs/libcuda.so -DCUDA_TOOLKIT_ROOT_DIR=/usr/local/cuda
4. make -j$(nproc)

Compiled plugin is provided in release page

How to use with CUDA

./bazadedicatedminer -u <your_baza_wallet_address> --cuda --cuda-loader=<path to CUDA plugin (xmrig-cuda.dll or libxmrig-cuda.so)>
